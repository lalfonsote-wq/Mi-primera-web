Laboratorio: App de Películas

🚀 Tu Misión

Vas a construir la pantalla principal de una aplicación de películas (tipo Netflix) usando HTML y CSS. Iremos paso a paso.

Preparación:

  1. Crea una carpeta en tu computadora llamada mi-app-peliculas.
  2. Dentro, crea un archivo llamado index.html.
  3. Crea otro archivo llamado estilos.css.

🦴 Fase 1: El Esqueleto

Abre tu archivo index.html y copia este código. Presta atención a la etiqueta <link>, esa es la magia que trae los iconos a tu proyecto.

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi App de Películas</title>
    <!-- 1. Enlazamos los iconos -->
    <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css" />
    <!-- 2. Enlazamos nuestro CSS -->
    <link rel="stylesheet" href="estilos.css">
</head>
<body>

    <!-- Tu código irá aquí -->

</body>
</html>

📱 Fase 2: El Menú Inferior

Vamos a crear el menú que siempre se queda pegado abajo. Copia esto dentro de las etiquetas <body> en tu index.html.

<!-- Menú de Navegación -->
<nav class="menu-inferior">
    <a href="#"><i class="fa-solid fa-house"></i></a>
    <a href="#"><i class="fa-solid fa-compass"></i></a>
    <a href="#"><i class="fa-regular fa-heart"></i></a>
    <a href="#"><i class="fa-regular fa-user"></i></a>
</nav>

🎨 Fase 2: El CSS del Menú

Abre estilos.css y pega esto. El truco aquí es position: fixed para anclar el menú al fondo.

body {
    background-color: #0c0f16;
    color: white;
    margin: 0;
}

.menu-inferior {
    background-color: #1a1e28;
    position: fixed; /* Lo deja pegado */
    bottom: 0;       /* Al borde inferior */
    width: 100%;
    display: flex;   
    justify-content: space-around;
    padding: 15px 0;
}

.menu-inferior a {
    color: #adb5bd;
    font-size: 24px;
}

🎞️ Fase 3: El Carrusel (HTML)

Vuelve a index.html. Agrega esto justo arriba del código de tu menú. Usaremos imágenes de prueba.

<section class="carrusel-contenedor">
    <h2>Próximas Películas</h2>
    
    <div class="fila-con-scroll">
        <img src="https://via.placeholder.com/250x150/ff4d4d/white?text=Peli+1" class="poster">
        <img src="https://via.placeholder.com/250x150/4da6ff/white?text=Peli+2" class="poster">
        <img src="https://via.placeholder.com/250x150/4dff4d/white?text=Peli+3" class="poster">
    </div>
</section>

👈👉 Fase 3: El Scroll (CSS)

En estilos.css, el secreto es overflow-x: auto, que permite deslizar hacia los lados.

.carrusel-contenedor { padding: 20px; }
.carrusel-contenedor h2 { font-size: 18px; }

.fila-con-scroll {
    display: flex;
    gap: 15px;
    overflow-x: auto; /* ¡Activa el scroll! */
    padding-bottom: 10px;
}

.poster {
    border-radius: 10px;
    flex-shrink: 0; /* Evita que se aplasten */
}

▶️ Fase 4: Reproducir Video

Añade este código en tu index.html, debajo del carrusel y arriba del menú. La etiqueta controls muestra el botón de play.

<section style="padding: 20px;">
    <h2>Tráiler de Estreno</h2>
    
    <video width="100%" controls>
        <source src="http://commondatastorage.googleapis.com/gtv-videos-bucket/sample/BigBuckBunny.mp4" type="video/mp4">
        Tu navegador no soporta videos.
    </video>
</section>

🏆 Tu Reto Final

¡Has terminado la estructura básica! Ahora te toca personalizarla:

  1. Copia y pega el bloque del carrusel HTML para crear una segunda fila llamada "Películas Nuevas".
  2. Busca en Google imágenes de pósters de películas reales.
  3. Reemplaza los enlaces https://via.placeholder... por los enlaces de tus imágenes reales en el atributo src="".

¡Buena Suerte!
